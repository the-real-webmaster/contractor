# 🛠 Contractor X — Adaptive Landing Page

Одностраничный адаптивный лендинг, созданный с использованием HTML5, CSS3 и JavaScript. Проект демонстрирует навыки верстки, включая интерактивное бургер‑меню, адаптивные блоки, слайдер проектов и отзывов, секции услуг, блог и футер.

## 🔗 Ссылка
[Посмотреть проект](https://the-real-webmaster.github.io/contractor/)

## 🛠 Технологии
- HTML5  
- CSS3  
- JavaScript  
- Swiper.js (слайдеры)  
- Адаптивный дизайн под разные устройства

## 📌 Особенности
- Бургер‑меню для мобильных экранов  
- Интерактивные слайдеры Projects и Testimonials  
- Секции About, Services, Quote, Quality, Projects, Clients, Blog  
- Чёткая семантическая структура и стилизация элементов
